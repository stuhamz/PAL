PAL Demonstration — Export-Edge Fingerprint Drift

This page deterministically renders to Canvas 2D and WebGL, exports bytes, and computes SHA‑256 hashes. Turn your PAL extension OFF/ON and observe hash changes and hook diagnostics.

Environment

PAL diag: unknown

Canvas 2D Test

Draws a deterministic pattern seeded from location.href and time quantum; exports both getImageData and toDataURL for hashing.

WebGL Test

Renders a seeded fragment shader, reads back pixels with readPixels, and hashes the resulting byte array.

PAL Diagnostics

If the extension exposes a diagnostics object on window, it is shown below. Field names are read‑only; availability depends on your build.

How to use this page in a demo
  1. Load page with PAL OFF. Click Re-run tests, then Store as baseline.
  2. Enable PAL and reload. Click Re-run tests, then Compare with baseline.
  3. Expected: Canvas and (when readback engaged) WebGL hashes change. Diagnostics show hook counts.

No external libraries. All hashing uses crypto.subtle.digest('SHA-256'). Baseline is saved in localStorage['pal-baseline'].
